# **App Name**: HomeSphere

## Core Features:

- User Authentication: Register and log in users with JWT-based authentication.
- Device Management: List, control, and view status of smart home devices.
- Room Management: Organize devices by rooms (e.g., Living Room, Bedroom).
- Automation Rules: Create, edit, and delete automation rules based on time and device status.
- Activity Logs: Display recent activity logs showing device actions and triggers.
- Admin Panel: Manage devices and user roles with admin privileges.
- Voice Control: Use browser SpeechRecognition to understand and execute voice commands.
- Energy Consumption Dashboard: Show device power usage, total energy consumption, and estimated bill. LLM tool assists user by informing the user about energy usage

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey stability and trust in the automation system.
- Background color: Light gray (#F5F5F5), a very light desaturated shade of the primary hue, providing a clean, modern backdrop.
- Accent color: Purple (#7E57C2), an analogous hue to the primary color with increased brightness and saturation, is used for interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif font.
- Use clear, consistent icons for devices (light, fan, AC) and actions.
- Dashboard should feature a responsive, grid-based layout.
- Use subtle animations for state changes (e.g., toggling devices) to provide clear visual feedback.
